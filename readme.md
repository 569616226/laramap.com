## Browser support

* Chrome *(latest 2)*
* Edge *(latest 2)*
* Firefox *(latest 2)*
* Internet Explorer 10+
* Opera *(latest 2)*
* Safari *(latest 2)*

*This doesn't mean that Laramap cannot be used in older browsers, just that we'll ensure compatibility with the ones mentioned above.*
